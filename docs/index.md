# Quant AI

**Generative models in quant finance: what they are, what they buy you, and how to evaluate them.**

This book carries evidence and working models: executed notebooks on current
market data, with code folded behind the figures. The current module is
**Generative Scenario Analysis**. The main argument runs through four
chapters, ending with <a href="discussions/quant_ai/Failure%20Modes%20and%20Next%20Tests.html">Failure Modes and Next Tests</a>;
the detailed builds live under Demos, and paper-by-paper notes under
Featured Research.

## How to read

- **Figures carry the argument; code is folded.** Open any code cell to inspect the construction.
- **Every figure says where its information comes from.** Three provenance tiers, labeled in each source line: **owned** (built end-to-end on our data), **their code, our data** (published code run verbatim on our data), and **redrawn** (a paper's verified numbers in our visual system; we defend the evaluation design, never the estimate).
- **Baselines are real contenders.** Replay, bootstrap, PCA shocks, and GARCH are what a good desk already runs. Nothing deep earns a role without beating them on a decision metric.
